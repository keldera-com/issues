# 🐛 Keldera Barrel Tracker - Issue Tracker

**Report bugs and request features for [Keldera Barrel Tracker](https://app.keldera.com)**

---

## What is Keldera Barrel Tracker?

Keldera Barrel Tracker is a web application for tracking and managing barrels. Visit [keldera.com](https://keldera.com) to learn more.

---

## How to Submit an Issue

1. Click the **[Issues](../../issues)** tab above
2. Click **New Issue**
3. Choose the appropriate template:
   - 🐛 **Bug Report** - Something isn't working as expected
   - 💡 **Feature Request** - Suggest a new feature or improvement
4. Fill in as much detail as possible and submit

> Anyone can submit an issue - no account required beyond a free GitHub account.

---

## Before You Submit

Please take a moment to:

- **Search existing issues** to avoid duplicates
- **Check the [roadmap](https://github.com/keldera/roadmap)** to see if your idea is already planned
- Include screenshots or screen recordings where helpful

---

## Issue Labels

| Label | Description |
|-------|-------------|
| `bug` | Something is broken or not working correctly |
| `feature-request` | A new feature or enhancement suggestion |
| `question` | A question about how something works |
| `duplicate` | This issue already exists |
| `under review` | Being evaluated by the team |
| `in progress` | Actively being worked on |
| `closed` | Resolved or declined |

---

## Response Times

We aim to respond to all issues within **3–5 business days**. Critical bugs affecting core functionality will be prioritised.
